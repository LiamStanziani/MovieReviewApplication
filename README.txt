To have this run, go into the command prompt located in the root directory of the project and type npm install to install all of the packages
needed for this application. If you would like to download the apk for this application and run it on your phone, go to the 
mobile/platforms/android/app/build/outputs/apk/debug folder and place the app-debug.apk into your phone by using a usb connector or by
using a cloud service. After getting it, you can run the apk and it will install onto your phone.
